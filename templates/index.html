ZipState: A State Data API

A free API for querying state data from a given US ZIP code.

Available Data:

  • ZIP Type
  • State Abbreviation
  • City
  • County
  • Timezone
  • ITS SOI Code
  • Area Code
  • Country
  • Latitude/Longitude
  • ITS Estimated Population

Returns either a JSON object or plain text.

API Endpoints:

  • /api/ Returns details of a random state.
  • /api/yourZIPCode Returns details about the give ZIP code in JSON.
  • /api/yourZIPCode Returns details about the give ZIP code in text (string).
