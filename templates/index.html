ZipState: A State Data API

A free API for querying state data from a given US ZIP code.

Available Data:

  • ZIP Type
  • State Abbreviation
  • City
  • County
  • Timezone
  • ITS SOI Code
  • Area Code
  • Country
  • Latitude/Longitude
  • ITS Estimated Population

Returns a JSON object.

API Endpoints:

  • /api Returns details of a random state.
  • /api/yourZIPCode Returns the state details about a given ZIP code in JSON.
